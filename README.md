# FakeStore React App

A simple React project using FakeStoreAPI.

## Pages

- Home – Welcome page with a button to view products
- Products – Shows all products
- Product Details – Shows one product’s info
- Add Product – Form to add a new product
- Edit Product – Form to update a product

## Tech Used

- React
- React Router
- Axios
- React Bootstrap

## Run the Project

npm install
npm run dev
